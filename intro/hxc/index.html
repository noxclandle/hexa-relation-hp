Chapter_01 // 物質への回帰

現代社会において、私たちのアイデンティティはあまりにも「希薄」になった。SNSのプロフィール、QRコード、デジタルウォレット。それらは便利ではあるが、そこに「手触り」はない。0と1の海に漂うあなたの断片は、スマートフォンの画面が消えた瞬間に、実体を持たない影へと変わる。

HXC（Hexa Card）はこの流れに逆行する。

私たちは、高度な暗号化技術を宿した「物理的なカード」に、あなたのデジタルな魂を封じ込める。それは、かつての貴族が用いた「封蝋」であり、武士が腰に差した「刀」と同じ重みを持つ。相手の端末にタップするその一瞬、あなたのアイデンティティは物理世界を媒介にして、相手の意識へと「透過」されるのだ。

「便利さ」のために捨て去られた、対面における「衝撃」と「重厚感」。それを最新のテクノロジーで再定義することが、HXCの第一の使命である。

Chapter_03 // 真実のアーキテクチャ

HXCの内部構造を語る上で、避けて通れないのが「妥協のなさ」である。

一般的なデジタル名刺サービスは、利便性のためにセキュリティを犠牲にし、テンプレートのために個性を犠牲にしている。しかし、MENSAの脳が設計したHXCは、そのどちらも許容しない。

Next.js 14のApp Routerによる高速な体験、Prismaによる堅牢なデータ整合性、そしてTailwind CSSによる極限まで削ぎ落とされたUI。これらは単なる技術選定ではない。あなたのアイデンティティを1ミリ秒の遅延もなく、1ピクセルの狂いもなく相手に伝えるための「必然」である。

私たちは、NFCチップ（NTAG215）の物理的な限界にまで挑んでいる。チップのUID（固有識別子）を取得し、それを正規化した上でデータベースと照合する。この泥臭くも精密な作業の積み重ねが、コピー不可能な「本物の物理アイデンティティ」を創り出すのだ。

発売前、私たちは牙を研いでいる。

現在、HXCは「限定的なプロトタイプ」のフェーズにある。量産体制に入る前のこの静寂の中で、私たちはさらなるロジックの追加を行っている。 例えば、AR（拡張現実）を用いた名刺からのデータ噴出。例えば、組織全体の繋がりを可視化するグラフ・データベースとの統合。 私たちが目指すのは、名刺を売ることではない。ビジネスという名の「ゲーム」における、標準的なプロトコル（議定書）を確立することだ。
